Buffer UI for ArcGIS API for JavaScript
=======================================

Provides a UI for a buffer operation.

See it in action here: [Demo](http://wsdot-gis.github.io/arcgis-js-buffer/)

Install
-------

```console
npm install --save @wsdot/arcgis-buffer-ui
```
